# Children's Ministry Attendance Tracker 
A web app built with Flask and SQLite to track children's church attendance across age group classes. 
## Features 
- Take attendance by session date
- Manage attendance roster with class groups 
- View attendance history grouped by date and class 
- Export attendance to CSV 
- Password protected with login page upon opening the website 
## Setup 
1. To install: 'pip install flask'
2. Run the app: 'python app.py'
3. Open: 'http://127.0.0.1:5000' in your browser
## Classes the Website Tracks
- God's Heritage (0-6)
- Chosen Generatin (6-10)
- Royal Preteens (11-13)
- Teen Church (13-18)
